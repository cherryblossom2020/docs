Retrieves a list of SCP:SL servers available via the cache.

You can filter results using query parameters matching server properties (like `isoCode=US` or `modded=false`), limit the number of results with `limit`, select specific fields using `fields`, and sort the closest servers first using `sortBy=distance`.
