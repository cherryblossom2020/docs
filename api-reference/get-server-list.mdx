---
title: "Get Server List"
description: "Retrieves a list of SCP:SL servers with filtering, sorting, and field selection."
api: "GET https://nwapi.daon.xyz/"
---

Retrieves a list of SCP:SL servers available via the cache.

### Query Parameters

<ParamField query="fields" type="string">
  Optional. Comma-separated list of server object fields to include in the response. If omitted, all fields are returned.
  _Example: _
</ParamField>

<ParamField query="limit" type="integer">
  Optional. Maximum number of server results to return. Defaults to returning all matching servers.
  _Example: _
</ParamField>

<ParamField query="sortBy" type="string">
  Optional. Field to sort results by. Currently supports `distance` (sorts servers nearest to the client first, requires Cloudflare GeoIP). If omitted or invalid, no specific sorting is applied beyond the source order.
  _Example: _
</ParamField>

<ParamField query="{dynamic_filter}" type="string | boolean | number">
  Optional. **Dynamic Value Filter.** Any query parameter _not_ named `fields`, `limit`, or `sortBy` acts as a filter. The parameter name must match a field in the server data (e.g., `isoCode`), and the value is the desired value.
  Filters match case-insensitively for strings. Boolean filters accept `true` or `false`. Numeric filters match exact numbers. You can apply multiple filters.
  _Examples: , , , _
</ParamField>

### Responses

The response body is a JSON array containing server objects. The `X-Data-Timestamp` header in the response indicates the freshness of the data (Unix milliseconds).
