Retrieves a list of SCP:SL servers available via the cache. This endpoint allows for detailed querying to find specific servers.

### Query Parameters

<ParamField query="fields" type="string">
  Comma-separated list of server object fields to include in the response. If omitted, all fields are returned.
  \_Example: \_
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of server results to return. If omitted, all matching servers are returned. Must be a positive integer.
  \_Example: \_
</ParamField>

<ParamField query="sortBy" type="string">
  Field to sort results by. Currently supports `distance` (sorts servers nearest to the client first, requires Cloudflare GeoIP).
  \_Example: \_
</ParamField>

<ParamField query="{dynamic_filter}" type="string | boolean | number">
  **Dynamic Value Filter:** You can filter by almost any field present in the server data by using the field name as the query parameter.

  - String comparisons are case-insensitive (e.g., `isoCode=DE`).
  - Boolean comparisons use `true` or `false` (e.g., `modded=true`).
  - Numeric comparisons match exact numbers (e.g., `port=7777`).
  - Multiple filters can be combined (e.g., `?isoCode=US&friendlyFire=false`).
    \_Examples: , , , \_
</ParamField>

The response is a JSON array containing server objects that match the query criteria. The `X-Data-Timestamp` header indicates the freshness of the data.

<RequestExample>

```bash Example Request (Filter US, Non-Modded, Limit 5, Specific Fields)
curl --location --request GET 'https://nwapi.daon.xyz/?isoCode=US&modded=false&limit=5&fields=ip,port,players,info,isoCode'
```


```bash Example Request (Sort by Distance, Limit 10)
curl --location --request GET 'https://nwapi.daon.xyz/?sortBy=distance&limit=10'
```
