This endpoint retrieves the cached list of SCP: Secret Laboratory servers.

### Key Features:

*   **Filtering:** Use query parameters matching server properties (e.g., `?isoCode=US&modded=false`). Any parameter not named `fields`, `limit`, or `sortBy` acts as a filter. String matches are case-insensitive.
*   **Field Selection:** Use the `fields` parameter (e.g., `?fields=ip,port,players`) to get only the data you need.
*   **Limiting:** Use the `limit` parameter (e.g., `?limit=10`) to control the number of results.
*   **Sorting:** Use `sortBy=distance` to order servers from nearest to farthest based on your IP address location (requires Cloudflare GeoIP).
